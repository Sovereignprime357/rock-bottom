# Graphics Phase 2 — operator-eye acceptance set

This directory binds the Graphics Phase 2 visual decision to production commit
`bfca7431cad0363d20c6cbf5cd2e4c0157ea914c` and the exact Phase 1 baseline
`fdd0ccc364540cc46d67b14be51a5f9fb145fc36`.

Start with `full-map-before-after-day.png` and `full-map-before-after-night.png`. Each compares the
same 110-camera, gapless 8,600×5,600 world sweep at a fixed visual clock and random seed. The six
`detail-crops-*` sheets preserve 1:1 pixels for northwest, Skid Row, canal, southwest, south rail,
and Throne Ditch. `character-variant-atlas.png`, `grime-static-atlas.png`, and
`secondary-loop-strip.png` isolate the art families from the map.

The eye gate is deliberately still human. The permanent suite proves structure, determinism,
palette discipline, culling, and frozen gameplay; it cannot prove that the dirt looks good.
Particular weak reads to judge are the wire tangle, bent needle, spent stub, compressed horse cop,
and Mayor's Cousin silhouette. The manifest records the complete fixture, dimensions, hashes, and
browser performance evidence.

The temporary browser capture harness is not production code and is not shipped.
